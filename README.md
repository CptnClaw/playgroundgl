# playgroundgl
Playground for learning OpenGL

## Dependencies
- OpenGL 3.3
- GLFW
- GLAD
- GLM
- assimp

## Keyboard actions
- 1: Cycle rendering mode (default, wireframe, envmap reflection, envmap refraction)
- 2: Toggle flashlight.
- 3: Toggle sun.
- 4: Toggle object selection mode.
- 5: Cycle skybox.
